## 说明

安装本插件可以让你的频道机器人使用云崽的功能！^\_^

## 安装

```
 git clone --depth=1 https://github.com/2y8e9h22/QQGuild-Plugin.git ./plugins/QQGuild-Plugin
```

```
 pnpm install -P
```

## 配置

```YAML
# 频道机器人的 APPID
appId: ""
# 频道机器人的 TOKEN
token: ""
# 沙盒测试
sandBox: false
# 接收全部消息，公域机器人必须设置为 false
allMsg: false
# 直接启动
autoStart: false
```
